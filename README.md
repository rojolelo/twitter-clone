<h1>Twitter-Clone</h1>

  <h3>Made by Ronny Legones using: React, React-Router, and Firebase.</h3>
  
  <b>Features: </b>
  - Write and Send Tweet
  - Mentions
  - Notifications
  - Like/Dislike Tweet
  - Retweet
  - Search
  - Follow/Unfollow
  - Change Profile Picture
  - Change Cover Picture
  - Delete Tweet
  
  <b>Future Changes:</b>
  - Direct Messages
  - Comment Tweet
  - Edit Tweet
